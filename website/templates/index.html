Flight Generation Route Generation

Welcome to the flight simulation route generation tool.

This tool generates multi-flight journeys between the departure and arrival airports by two criteria.

Airlines Routes generates a journey using real-world airline routes. It takes the source and destination airports as inputs, as well as available aircraft types.

Routes by Range generats the shortest journey based on the input criteria. This takes the source and destination airports as inputs, as well as maximum aircraft range and the minimum runway length.
